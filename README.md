# mckinseyhpf

## Getting started

## Frontend

In ./frontend

1. Install npm packages
   `npm i`
2. Run npm
   `npm start`

## Backend

In ./backend

1. Install and Activate env

    ```
    python3 -m venv env
    source env/bin/activate
    ```

2. Install Pip requirements
   `pip install requirements.txt`

3. Run Flask
   `python main.py`
