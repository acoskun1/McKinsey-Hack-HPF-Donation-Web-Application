# mckinseyhpf

## Getting started

## Frontend

In ./frontend

1. Install npm packages
   `npm i`
2. Run npm
   `npm run dev`
3. Open 'http://localhost:3000' in browser

## Backend

In ./backend

1. Install and Activate env

    ```
    python3 -m venv env
    source env/bin/activate
    ```

2. Run Flask
   `python main.py`
